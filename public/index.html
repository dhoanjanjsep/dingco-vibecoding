🎨 Cursor Talk to Figma MCP
AI와 Figma를 연결하는 Model Context Protocol 통합

프로젝트 소개

Cursor Talk to Figma MCP는 Cursor AI와 Figma 간의 원활한 통신을 가능하게 하는 Model Context Protocol (MCP) 통합 도구입니다. AI를 통해 Figma 디자인을 읽고, 수정하고, 자동화할 수 있습니다.

🔧 설치 및 설정

Windows 사용자를 위한 원클릭 설치:

npm run install-and-setup

🎯 주요 기능

  • Figma 디자인 읽기
  • 요소 생성 및 수정
  • 텍스트 일괄 변경
  • 컴포넌트 관리

🚀 시작하기

  1. WebSocket 서버 시작
  2. Figma 플러그인 설치
  3. Cursor와 연결

링크

GitHub 저장소 Figma 플러그인

설치 가이드

Windows 사용자

가장 간단한 방법:

npm run install-and-setup

macOS/Linux 사용자

curl -fsSL https://bun.sh/install | bash
bun setup
